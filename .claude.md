# 专利管理器项目 - AI协作规范

## 一、项目背景

这是一个为**企业专利工程师**设计的单机版桌面应用。

**核心功能**：
- 专利全生命周期管理（申请→审查→授权→年费维护→终止）
- 年费自动计算（含滞纳金）
- 状态自动流转
- 资产统计分析

**技术栈**：
- Electron 28+（桌面框架）
- better-sqlite3（数据库）
- 原生 JavaScript（不引入Vue/React）
- ECharts（图表）

**用户画像**：非技术背景的企业专利工程师。代码必须易理解、易维护。

### 规则4：复杂逻辑逐行注释

对于超过5行的循环、条件判断、计算逻辑，每一行都要注释。

示例:
// 函数：计算滞纳金
function calculatePenalty(dueDate, paidDate) {
    // 第1步：计算逾期天数
    const due = new Date(dueDate);
    const paid = new Date(paidDate);
    // 第2步：计算逾期月数（不足一个月按一个月算）
    let monthsOverdue = (paid.getFullYear() - due.getFullYear()) * 12;
    monthsOverdue += paid.getMonth() - due.getMonth();
    // 第3步：如果逾期天数超过15天，多算一个月
    if (paid.getDate() - due.getDate() > 15) {
        monthsOverdue += 1;
    }
    // 第4步：限制最大逾期月数为6（超过6个月专利失效）
    monthsOverdue = Math.min(monthsOverdue, 6);
    
    // 第5步：滞纳金比例对照表（逾期1个月5%，2个月10%...6个月25%）
    const penaltyRates = {
        0: 0,    // 未逾期
        1: 0.05,
        2: 0.10,
        3: 0.15,
        4: 0.20,
        5: 0.25,
        6: 0.25
    };
    
    // 第6步：返回滞纳金比例
    return penaltyRates[monthsOverdue] || 0;
}
