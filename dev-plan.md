# 企业专利全生命周期管理系统 - 开发计划

> 此文档记录项目所有已实现功能，供后续开发和维护参考。

---

## 一、项目脚手架

- [x] 初始化项目（package.json、sql.js + ECharts + xlsx）
- [x] Node.js 原生 HTTP 服务器（server.js，无 Express）
- [x] sql.js WASM SQLite 数据库层（database.js）
- [x] 前端 API 适配层（api.js / preload.js，双模式支持 web + Electron）
- [x] CSP 安全策略（'unsafe-eval' 用于 ECharts）

## 二、数据库

- [x] 11 张业务表：settings / patents / fee_tasks / attachments / operation_logs / fee_standards / fee_reduction_rates / penalty_rates / status_transitions / fee_due_rules / pending_urgent_tasks
- [x] 种子数据：15 条年费标准 + 12 条状态流转 + 5 档费减 + 7 档滞纳金 + 5 条费用截止规则
- [x] 首次启动自动导入种子数据 + 自动备份

## 三、登录 & 安全

- [x] 首次使用密码设置页（密码 + 确认密码 + 密保问题）
- [x] 登录页（密码输入 + 解锁 + 错误提示）
- [x] SHA-256 哈希密码存储
- [x] "忘记密码"链接（始终显示）
- [x] 密保问题找回流程：展示问题 → 验证答案（小写去空格后哈希比对）→ 设置新密码
- [x] 密保问题设置：5 个预设问题 + 自定义自由输入
- [x] 安全设置弹窗（侧边栏底部齿轮图标，登录后可修改密保）
- [x] 登录冷却：3 次失败后倒计时 30s，4 次 60s，最多 120s
- [x] 冷却时引导使用"忘记密码"
- [x] 老用户兼容（未设密保时不显示忘记密码流程）

## 四、数据中心

- [x] 子标签导航（Excel 导入 / 导出 / 回收站）
- [x] Excel 导入：模板下载（含待办任务/截止日/费用列 + 填写说明）、表头自动匹配、预览前 20 条、错误报告
- [x] 导入去重：文件内去重 + 数据库去重弹窗（跳过/覆盖/取消）
- [x] 导入时自动创建费用任务 + 自动计算金额 + 自动备份
- [x] 导入后跨页面同步（工作台自动刷新）
- [x] Excel 导出：10 个字段多选 + 中文表头映射
- [x] 回收站：查看 / 单条恢复 / 单条彻底删除 / 批量恢复 / 批量彻底删除 / 30 天自动清理
- [x] 数据库快照备份（导入前自动触发，保留最近 5 次，备份列表查看）

## 五、工作台 — 列表 & 检索

- [x] 专利列表表格（复选框 + 预警灯 + 专利号/名称/类型/状态/紧迫任务/操作）
- [x] 复合检索：关键词（专利号/名称/发明人）+ 类型 + 状态 + 预警 + 日期范围
- [x] 日期类型分段控件（申请日 / 截止日 / 授权日）
- [x] 可折叠筛选面板（类型/状态/预警下拉，滑入动画）
- [x] 分页（页码 / 上下页 / 省略号 / 每页 10/15/30/50 条）
- [x] 状态标签 4 色：灰色（撰写中）、蓝色（审查中）、绿色（生效）、红色（终止）
- [x] 紧迫任务展示：费用 > 手动待办 > 缺附件 > 待确认（简单顺序）
- [x] 预警灯 4 色：红色（逾期天数）、黄色（剩余天数）、橙色（附件）、绿色（正常）
- [x] 批量操作：批量删除、批量完成缴费

## 六、工作台 — 状态流转引擎

- [x] 状态机核心逻辑（依据 STATUS_TRANSITIONS 表驱动）
- [x] 6 种状态变更路径：→已申请、→通知授权、→终态、→专利权生效 等
- [x] 任务自动生成：申请费/实审费/年费（首次在状态变更时创建，后续缴费后自动续期）
- [x] 年费自动续期：缴费后生成下一年年费
- [x] 发明 vs 实用新型/外观设计 分支逻辑（形式审查中分流）
- [x] 状态流转含"待实质审查"（形式审查中 → 待实质审查 → 实质审查中）

## 七、工作台 — 完成缴费弹窗

- [x] 动态渲染：状态流转卡片 + 独立缴费任务列表
- [x] 流转卡片：附件上传（上传后标记完成）、缴费确认复选框、执行状态变更
- [x] 独立缴费任务：年费等不触发流转的费用，勾选确认
- [x] 年费缴费后自动创建下一年任务 + 操作日志
- [x] 操作前自动备份

## 八、工作台 — 专利详情页

- [x] 右侧抽屉浮层（40% 宽度，4 个标签页）
- [x] 基本信息：3 列网格 + 状态流管道图（水平节点+连接线）+ 编辑模式
- [x] 编辑模式：专利信息编辑 + 紧迫任务管理（费用/非费用待办）
- [x] 非费用待办下拉：9 个预设选项按流程顺序排列（提交申请/受理通知书/初审合格/实审通知书/OA/意见陈述书/授权通知书/驳回/撤回）
- [x] 缴费记录列表
- [x] 操作日志列表（最近 50 条）
- [x] 附件管理：上传（20MB 限制，仅 PDF/PNG/JPG/GIF 类型选择）+ 预览 + 删除

## 九、仪表盘（ECharts 可视化）

- [x] 资产构成饼图（环形图：发明/实用新型/外观设计，tooltip 含发明人排名）
- [x] 动态存量柱状图（堆叠柱状：深色=有效数、浅色=申请数，近 1/2/3 年切换）
- [x] 柱状图自定义 HTML 图例（3 列 × 2 行，点击切换系列显隐）
- [x] 统计概览卡片 4 张（专利总数 + 已逾期 + 待办 + 本月应缴）
- [x] 费用预算视图：起始/截止月份查询（支持跨年）+ 按费用类型汇总
- [x] 费用明细弹窗：点击"查看详情"钻取逐条记录（专利号/名称/年度/金额/截止日/状态）
- [x] 数据完整性：EXISTS 子查询过滤孤立数据
